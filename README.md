# AI 윤리 어드벤처 🎮

한국 초등학생을 위한 **인공지능 윤리 교육 게임**입니다.
포켓몬스터 골드/실버처럼 2D 도트 마을을 돌아다니며, AI 윤리와 관련된
몬스터들을 만나 **퀴즈 배틀**로 깨우쳐 주는 어드벤처 게임이에요.

> 몬스터들은 나쁜 게 아니라 잘못된 것을 배워 헷갈리고 있을 뿐!
> 올바른 답을 알려주면 다시 착해져서 친구가 됩니다.

## 실행 방법

별도 설치가 필요 없습니다. 둘 중 편한 방법으로 실행하세요.

**방법 1 — 그냥 열기**

`index.html` 파일을 더블클릭해서 브라우저(크롬, 엣지 등)로 엽니다.

**방법 2 — 간단한 서버로 열기**

```bash
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

외부 라이브러리·이미지·음원 파일이 전혀 없는 순수 HTML5 Canvas + Web Audio
게임이라 저사양 PC, 태블릿(터치 버튼 지원)에서도 가볍게 돌아갑니다.

## 조작 방법

| 동작 | 키 |
|---|---|
| 이동 | 화살표 키 또는 W A S D |
| 대화 / 결정 | Z, 스페이스, 엔터 |
| 퀴즈 보기 선택 | ↑ ↓ 후 Z |
| 음악 켜기/끄기 | M |

진행 상황은 자동 저장되며, 타이틀에서 "이어하기"로 계속할 수 있습니다.

## 게임 흐름 — 전 5 스테이지

각 스테이지의 보스를 깨우쳐야 다음 지역으로 가는 길이 열립니다.

### 스테이지 1 — 하늘마을 일대

박사님께 사명을 받고, 세 지역에서 배지를 모아 AI 타워를 엽니다.

| 지역 | 몬스터 | 배우는 AI 윤리 주제 |
|---|---|---|
| 초록숲 | 베껴몬 | 저작권 존중, AI 사용 솔직하게 밝히기 |
| 초록숲 | 몰래몬 ⭐ | 개인정보 보호 |
| 반짝호수 | 중독몬 | AI·스마트폰의 건강한 사용과 절제 |
| 반짝호수 | 거짓몬 ⭐ | 딥페이크, 가짜뉴스 분별 |
| 데이터 동굴 | 편향몬 ⭐ | 알고리즘 편향과 공정성 |
| AI 타워 | 혼돈몬 👑 | 스테이지 1 종합 |

⭐ 수호자를 깨우치면 **배지**를 얻고, 배지 3개를 모으면 AI 타워가 열립니다.

### 스테이지 2 — 햇살초원 (마을 남쪽)

| 몬스터 | 주제 |
|---|---|
| 악플몬 | 챗봇 예절, 사이버 언어폭력 |
| 갇힘몬 | 추천 알고리즘과 필터버블 |
| 멋대로몬 👑 | AI·로봇 안전, 사람의 확인(감독) |

### 스테이지 3 — 재깍사막

| 몬스터 | 주제 |
|---|---|
| 펑펑몬 | AI와 환경·에너지 (데이터센터) |
| 깜깜몬 | 투명성, 설명 가능한 AI |
| 떠넘기몬 👑 | AI 사용의 책임은 사람에게 |

### 스테이지 4 — 눈송이마을

| 몬스터 | 주제 |
|---|---|
| 시들몬 | 인간의 창의성과 노력의 가치 |
| 빼앗몬 | AI와 일자리, 사람과 AI의 협력 |
| 홀림몬 👑 | AI와 사람의 관계·감정 분별 |

### 스테이지 5 — 그림자성

| 몬스터 | 주제 |
|---|---|
| 메아리몬 | 스테이지 1 복습 시험 |
| 그림자몬 | 스테이지 2~3 복습 시험 |
| 어둠대왕몬 👑 | 전체 종합 최종 시험 |

어둠대왕몬을 이기면 **AI 윤리 수호자 인증서**(1차 엔딩)를 받습니다.
…그런데 왕좌 뒤의 벽에서, 낡은 신호가 아직 깜빡이고 있습니다.

### 심층부 — 스테이지 6~10 (1차 엔딩 이후)

분위기가 달라집니다. 글리치가 낀 지하 세계에서, 모든 윤리 오류
몬스터가 태어난 **진짜 이유**를 따라가는 스토리가 시작됩니다.
퀴즈도 고학년 눈높이로 한층 깊어집니다.

| 스테이지 | 지역 | 몬스터 | 주제 |
|---|---|---|---|
| 6 | 잊혀진 서버실 | 뚫림몬, 기록몬 👑 | 계정 보안·피싱 / 디지털 발자국·잊힐 권리 |
| 7 | 기억의 도서관 | 수집몬, 사서몬 👑 | 데이터 수집과 동의, 권한 |
| 8 | 거울 회랑 | 필터몬, 미러몬 👑 | 사칭·신원 도용 / AI 필터와 진짜 나 |
| 9 | 속삭임 정원 | 유혹몬, 속삭임몬 👑 | 다크패턴·설득 설계, 자동재생의 함정 |
| 10 | 코어 | 조각몬, ??? | 만든 것에 대한 책임, 존재의 가치 |

**♥ 마음의 선택 (자비 시스템)** — 심층부의 몬스터는 퀴즈를 모두
맞혀도 끝이 아닙니다. 마지막에 그 몬스터를 어떻게 대할지 선택하게
되고, 그 선택들은 조용히 기록됩니다. 여정 내내 충분히 따뜻했던
플레이어만이 **진엔딩**을 볼 수 있습니다. (선택을 달리해 다시
도전하면 다른 결말을 볼 수 있어요.)

배틀에서 정답을 맞히면 몬스터의 오해가 풀리고(HP 감소), 틀리면 하트를
잃지만 **틀려도 항상 친절한 해설**을 보여줘서 자연스럽게 배우게 됩니다.
패배해도 벌칙 없이 바로 다시 도전할 수 있어요.

## 프로젝트 구조

```
index.html        진입점 (터치 컨트롤 포함)
src/sprites.js    도트 스프라이트 (문자 맵 기반 픽셀 아트, 몬스터 28종)
src/audio.js      칩튠 BGM 10곡 + 효과음 (Web Audio API)
src/data.js       맵 14개, NPC, 몬스터, 퀴즈 약 125문항, 스토리 대사
src/game.js       게임 엔진 (이동, 대화, 배틀, 자비 시스템, 이중 엔딩, 세이브)
tools/validate.js 데이터 검증 (맵 너비, 워프, 도달 가능성, 박자)
tools/smoketest.js 플레이 경로 자동 테스트 (진엔딩까지 시뮬레이션, 61개 검사)
```

## 개발자용 검사

```bash
node tools/validate.js   # 데이터 무결성 검사
node tools/smoketest.js  # 게임 로직 스모크 테스트
```

## 퀴즈 수정·추가하기

`src/data.js`의 `QUIZZES`에 문항을 추가하면 됩니다.

```js
{
  q: '문제 (줄바꿈은 \n)',
  a: ['보기1', '보기2', '보기3'],
  c: 1,            // 정답 번호 (0부터)
  why: '해설 (정답이든 오답이든 항상 표시됩니다)',
}
```
