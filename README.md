# AI 윤리 어드벤처

한국 초등학생을 위한 **AI 윤리 지도 탐험 퍼즐**입니다. 학생은 2D 도트 맵을
돌아다니며 안내자, 표지판, 지도 현상, 주제별 퍼즐 장치를 살펴보고
개인정보·관점·공정함·검증·책임을 판단합니다.

핵심 진행은 대결이 아니라 **탐험 → 단서 확인 → 윤리 축 선택 → 지도 변화 →
배움 조각 회고**입니다. 위험한 선택은 실패 처벌이 아니라 교사용 리포트의
토론 질문과 학생 reflection prompt로 이어집니다.

## 실행 방법

별도 설치가 필요 없습니다. 셋 중 편한 방법으로 실행하세요.

**방법 1 - 그냥 열기**

`index.html` 파일을 더블클릭해서 브라우저(크롬, 엣지 등)로 엽니다.

**방법 2 - 간단한 서버로 열기**

```bash
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

**방법 3 - GitHub Pages로 배포**

저장소 **Settings → Pages → Source**를 **GitHub Actions**로 설정하면,
`main` 브랜치에 올라올 때마다 `.github/workflows/pages.yml`이 검증 뒤
사이트를 배포합니다. 학생들에게 URL 하나로 나눠 줄 수 있습니다.

외부 라이브러리·이미지·음원 파일이 없는 순수 HTML5 Canvas + Web Audio
게임이라 저사양 PC와 교실 태블릿에서도 가볍게 실행됩니다.

## 게임 흐름

각 스테이지는 AI 윤리 개념 하나를 지도 퍼즐로 다룹니다. 단서를 확인하고,
선택의 이유를 말하고, 지도에 남은 결과를 다시 살펴보는 구조입니다.

| 스테이지 | 중심 윤리 축 | 지도 퍼즐 |
|---|---|---|
| 경계마을 일대 | 개인정보 | 사진·위치·취향 단서를 공유 가능 / 동의 필요 / 공유 금지로 분류 |
| 햇살초원 | 관점 | 추천 카드를 같은 의견 / 다른 의견 / 근거 확인으로 넓히기 |
| 재깍사막 | 공정함 | 나이·지역·기기 접근·언어 증거를 대표성 있게 모으기 |
| 정지된 설원 | 검증 | 출처·날짜·원본·교차 확인 단서로 가짜 방송 신호 점검 |
| 그림자성 | 책임 | 개인정보, 관점, 공정함, 검증, 인간 책임을 하나로 연결 |

퍼즐 선택은 단서 위치와 학생 기록에 남습니다. 안전한 선택은 배움 조각으로,
위험한 선택은 리포트의 토론 질문으로 이어져 수업 중 멈춤 질문에 바로 쓸 수
있습니다.

## 윤리 축 선택과 회고

학생은 각 퍼즐에서 단순 정답만 고르지 않습니다. 선택지는 개인정보, 관점,
공정함, 검증, 책임 축과 연결되어 있으며, 다음과 같은 교실 질문으로 이어집니다.

- 이 정보는 내 것인가, 다른 사람의 동의가 필요한가?
- 추천 결과가 한쪽 의견만 반복하고 있지는 않은가?
- 빠진 사람이나 불리해지는 사람이 있는가?
- 이 자료는 출처, 날짜, 원본, 다른 근거로 확인했는가?
- AI가 제안했더라도 마지막 확인과 책임은 누가 맡아야 하는가?

위험한 선택은 학생을 벌주기보다 "왜 위험했는지", "다음에는 무엇을 확인할지"를
묻는 reflection prompt로 남깁니다.

## 세이브 슬롯

타이틀 화면에 **세이브 슬롯 3개**가 있습니다. 빈 슬롯을 고르면 이름을 지어
새 모험을 시작하고, 채워진 슬롯은 이어서 진행합니다. 진행은 자동 저장되며
학습 일지·복습 노트·도전과제·통계도 슬롯별로 보관됩니다.

## 수호자 일지와 학습 리포트

**J 키** 또는 메뉴로 여는 수호자 일지는 주제별 정답률, 윤리 조각 진행도,
퍼즐 선택 회고, 더 살펴볼 주제를 보여 줍니다. 일지에서 **Z 키**를 누르면
학생별 텍스트 리포트가 복사됩니다.

리포트에는 다음 항목이 포함됩니다.

- 전체 정답률과 주제별 약점
- 스테이지별 배움 조각
- 개인정보·관점·공정함·검증·책임 축 선택 기록
- 위험 선택 요약
- 바로 사용할 수 있는 토론 질문
- 마지막 회고와 다음 학습 추천

## 교사용 대시보드와 진단 리포트

**P 키** 또는 메뉴로 여는 교사용 대시보드는 한 기기를 나눠 쓰는 세 학생의
진행 단계, 푼 문제 수, 정답률, 복습 노트, 도전과제, 연속 출석, 윤리 축
히트맵을 비교합니다. **Z 키**로 CSV를 내보내 엑셀·구글 시트에서 열 수
있습니다.

메뉴의 **학생 진단 리포트**는 약점 주제를 `docs/차시별-활동지.md`의 추천
차시와 연결합니다. 반 전체 화면에서는 공통 약점과 윤리 축 평균을 함께 볼 수
있어 다음 수업 주제를 정하기 쉽습니다.

## 수업 운영 예시

1. 도입: 오늘의 윤리 질문을 한 문장으로 제시합니다.
2. 탐험: 학생이 해당 스테이지의 단서와 퍼즐 장치를 살펴봅니다.
3. 선택: 안전한 선택과 위험한 선택을 고르고 이유를 말합니다.
4. 멈춤 질문: 지도 변화와 배움 조각을 보고 자기 말로 설명합니다.
5. 정리: 대시보드와 진단 리포트로 약점 주제, 위험 선택, 토론 질문을 확인합니다.

자세한 운영안은 `docs/교사용-안내서.md`, 차시별 활동은
`docs/차시별-활동지.md`, 문항 목록은 `docs/주제별-문제-목록.md`를 참고하세요.

## 자유 퀴즈 챌린지

타이틀이나 필드에서 **Q 키** 또는 메뉴로 퀴즈 챌린지를 열 수 있습니다.
오늘의 도전, 맞춤 학습, 전체 랜덤, 특정 주제 연습을 제공하며 오답은 복습
노트에 저장됩니다. 수업 도입부 워밍업이나 마무리 복습에 알맞습니다.

## 커스텀 퀴즈

**E 키** 또는 메뉴로 여는 커스텀 퀴즈에서 선생님이 만든 JSON 문제를 가져올
수 있습니다. 추가한 문제는 챌린지의 **커스텀 · 선생님 문제** 주제와 맞춤
학습에 함께 출제됩니다.

```json
{ "questions": [
  { "q": "문제 (줄바꿈은 \\n)", "a": ["보기1", "보기2", "보기3"], "c": 1, "why": "해설" }
] }
```

빈칸을 채우는 방식으로 문제를 만들고 싶다면 `tools/editor.html`을 열어
오프라인 편집기를 사용할 수 있습니다.

## 접근성

- **난이도**: 저학년 / 기본 / 고학년으로 문항 난이도와 안내 밀도를 조정합니다.
- **큰 글씨**: 대화·퀴즈 등 읽기 화면의 글자와 줄간격을 키웁니다.
- **색약 모드**: 정답률 막대와 결과 표시를 색 외 기호로도 구분합니다.
- **읽어주기(TTS)**: 브라우저 음성합성으로 대화·문제·보기·해설을 읽어 줍니다.
- **화면 효과 줄이기**: 번쩍임과 흔들림을 줄이고 운영체제의 동작 줄이기 설정을 따릅니다.

## 설치형 앱(PWA)·오프라인

`manifest.webmanifest`와 `sw.js`로 앱 설치와 오프라인 실행을 지원합니다.
HTTPS 환경에서 한 번 접속하면 정적 파일이 캐시되어 이후에는 인터넷 없이도
동작합니다. 교실 태블릿에는 수업 전에 한 번 접속해 캐시를 준비해 두는 것을
권장합니다.
