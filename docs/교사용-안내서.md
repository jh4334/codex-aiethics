# 교사용 안내서 — AI 윤리 어드벤처

초등학생이 **몬스터의 마음을 안아 주며** 개인정보·저작권·공정함·생성형 AI 등
AI 윤리를 배우는 교육용 게임입니다. 설치가 필요 없고, 인터넷이 끊겨도 돌아갑니다.

- 규모: 맵 21개 · 몬스터 35종 · 퀴즈 28주제 173문항 · 배경음악 11곡
- 권장 환경: 가로 화면(태블릿·노트북·PC), 최신 크롬·사파리·엣지
- 한 기기를 **세 명(슬롯 1·2·3)** 이 나눠 쓸 수 있습니다.

---

## 1. 실행하는 세 가지 방법

1. **그냥 열기** — `index.html` 파일을 더블클릭하면 바로 시작됩니다(인터넷 불필요).
2. **수업용 서버** — 폴더에서 `python3 -m http.server` 실행 후
   `http://localhost:8000` 접속. 여러 대가 같은 와이파이로 들어올 때 편합니다.
3. **웹 + 앱 설치(PWA)** — GitHub Pages 같은 주소로 한 번 접속하면, 브라우저의
   "홈 화면에 추가 / 앱 설치"로 설치됩니다. 이후에는 **오프라인에서도** 실행됩니다.

> 처음 한 번만 인터넷에 연결되면, 그다음부터는 네트워크 없이도 수업이 가능합니다.

---

## 2. 조작법

| 동작 | 키보드 | 터치(태블릿) |
|------|--------|--------------|
| 이동 | 방향키 / WASD | 화면 왼쪽 십자 버튼 |
| 결정·대화 | Z / Space / Enter | 오른쪽 큰 A 버튼 |
| 메뉴·뒤로 | X / Esc | 왼쪽 위 "메뉴" |
| 도감 | C | 오른쪽 위 "도감" |
| 50:50 힌트(문제풀이 중) | H | "힌트" 버튼 |

터치 기기는 자동으로 화면 버튼이 나타납니다. 세로로 들면 "가로로 돌려 주세요" 안내가 뜹니다.

---

## 3. 선생님 전용 기능 (메뉴 → 일시정지 화면)

- **▤ 교사용 대시보드** — 세 학생(슬롯)의 진행 단계·푼 문제·정답률·복습 노트·
  도전과제·연속 출석을 한 화면에서 비교합니다. **`Z`를 누르면 반 전체 현황을
  CSV 파일로 내보내** 엑셀·구글 시트에서 열 수 있습니다(한글 깨짐 없음).
- **▶ 수업 모드 (스테이지 시작)** — 오늘 수업할 스테이지(1~10)를 골라 **바로 시작**
  합니다. 이전 스테이지는 완료 처리되고, 학생은 해당 스테이지 시작 지점에서 출발해요.
  지금 학생 슬롯의 진행을 바꾸므로, 실행 전 한 번 더 확인을 거칩니다.
  (되돌릴 수 없으니 미리 「데이터 백업」을 권장합니다.)
  · 특정 **주제만** 연습시키려면 「퀴즈 챌린지」에서 주제를 골라 출제하세요.
- **✎ 커스텀 퀴즈** — 선생님이 만든 문제를 파일이나 클립보드로 가져와,
  퀴즈 챌린지의 "커스텀 · 선생님 문제" 주제로 출제할 수 있습니다.
  문제 템플릿(JSON)도 이 화면에서 받을 수 있습니다.
  · **쉬운 문제 만들기** — `tools/editor.html` 파일을 두 번 눌러 열면(인터넷 불필요),
    빈칸을 채우는 방식으로 문제를 만들고 정답·해설·글자 수까지 확인한 뒤
    JSON으로 저장/복사할 수 있어요. 그 파일을 위 "가져오기"로 넣으면 됩니다.
- **🏆 수료증** — 학생별 학습 결과를 담은 인증서 화면. 캡처하거나 인쇄해
  과제·포트폴리오로 활용하세요.
- **⇄ 데이터 백업·복원** — 모든 슬롯·기록·설정을 한 파일로 저장하고 다른
  기기에서 복원합니다. 기기 교체나 학기말 보관에 쓰세요.

## 4. 접근성 (저학년·특수교육 배려)

일시정지 메뉴에서 켤 수 있습니다.

- **난이도**: 저학년 / 기본 / 고학년 — 회피 난이도와 문항 난이도가 함께 조정됩니다.
- **자막 속도**: 느림 / 보통 / 빠름
- **큰 글씨**, **색약 모드**, **읽어주기(TTS)**, **화면 효과 줄이기**(광과민성 배려)

---

## 5. 수업에 쓰는 흐름(예시)

1. 도입 — 오늘 다룰 주제를 한 문장으로 소개(예: "AI도 틀릴 수 있을까?").
2. 플레이 — 해당 스테이지를 15~20분 진행. 몬스터를 "이기기"보다
   **마음을 안아 주는 선택**을 고민하게 합니다.
3. 정리 — 대시보드에서 반 정답률을 보고, 약점 주제를 함께 토론합니다.
4. 확장 — `docs/차시별-활동지.md`의 토론·활동지를 활용합니다.

전체 문항은 `docs/주제별-문제-목록.md`에서 한눈에 볼 수 있습니다(자동 생성).

## 6. 자주 묻는 질문

- **검은 화면만 나와요** — 처음 로딩에 실패한 경우 "다시 시도" 버튼이 보입니다.
  인터넷 연결을 확인하고 새로 고침하세요.
- **기록이 사라졌어요** — 같은 브라우저·기기에서만 저장됩니다(시크릿 모드 주의).
  미리 "데이터 백업"으로 파일을 받아 두면 안전합니다.
- **문제를 우리 반에 맞게 바꾸고 싶어요** — "커스텀 퀴즈"로 직접 추가하거나,
  개발자가 `src/data.js`의 `QUIZZES`를 수정한 뒤 `node tools/quizlist.js`로
  목록 문서를 다시 만들 수 있습니다.
