# 교사용 안내서 — AI 윤리 어드벤처

초등학생이 **지도 탐험 퍼즐**을 풀며 개인정보·관점·공정함·검증·책임을
AI 윤리 언어로 정리하는 교육용 게임입니다. 설치가 필요 없고, 인터넷이 끊겨도 돌아갑니다.

- 구성: 맵 탐험 · 주제별 퍼즐 장치/안내자 · 퀴즈 28주제 175문항 · 배경음악 11곡
- 권장 환경: 가로 화면(태블릿·노트북·PC), 최신 크롬·사파리·엣지
- 한 기기를 **세 명(슬롯 1·2·3)** 이 나눠 쓸 수 있습니다.

---

## 1. 실행하는 세 가지 방법

1. **그냥 열기** — `index.html` 파일을 더블클릭하면 바로 시작됩니다(인터넷 불필요).
2. **수업용 서버** — 폴더에서 `python3 -m http.server` 실행 후
   `http://localhost:8000` 접속. 여러 대가 같은 와이파이로 들어올 때 편합니다.
3. **웹 + 앱 설치(PWA)** — GitHub Pages 같은 주소로 한 번 접속하면, 브라우저의
   "홈 화면에 추가 / 앱 설치"로 설치됩니다. 이후에는 **오프라인에서도** 실행됩니다.

> 처음 한 번만 인터넷에 연결되면, 그다음부터는 네트워크 없이도 수업이 가능합니다.

## 2. 조작법

| 동작 | 키보드 | 터치(태블릿) |
|------|--------|--------------|
| 이동 | 방향키 / WASD | 화면 왼쪽 가상 스틱(끌어서 이동) |
| 결정·대화 | Z / Space / Enter | 오른쪽 큰 A 버튼 |
| 메뉴·뒤로 | X / Esc | 왼쪽 위 "메뉴" |
| 주제 기록 | C | 오른쪽 위 "기록" |

터치 기기는 자동으로 화면 버튼이 나타납니다. 세로로 들면 "가로로 돌려 주세요" 안내가 뜹니다.

## 3. 선생님 전용 기능 (메뉴 → 일시정지 화면)

- **▤ 교사용 대시보드** — 세 학생(슬롯)의 진행 단계·푼 문제·정답률·복습 노트·
  도전과제·연속 출석·5개 윤리 축 히트맵을 한 화면에서 비교합니다. **`Z`를 누르면
  반 전체 현황과 윤리 축 점수를 CSV 파일로 내보내** 엑셀·구글 시트에서 열 수
  있습니다(한글 깨짐 없음).
- **🩺 학생 진단 리포트** — 학생(슬롯)의 약점 주제를 찾아 **추천 차시와 자동
  연결**해 줍니다. ◀▶로 학생을 바꾸거나 **「반 전체」 공통 약점**(학생 수 많은
  순)과 윤리 축 평균을 보고, `Z`로 인쇄·보관용(.txt/클립보드)으로 내보내 다음
  수업 주제를 바로 정할 수 있어요.
- **▶ 수업 모드 (스테이지 시작)** — 오늘 수업할 본편 스테이지(1~5)를 골라 **바로 시작**
  합니다. 이전 스테이지는 완료 처리되고, 학생은 해당 스테이지 시작 지점에서 출발해요.
  화면에 스테이지 이름·윤리 축·핵심 배움 문장이 함께 표시됩니다. 지금 학생 슬롯의
  진행을 바꾸므로, 실행 전 한 번 더 확인을 거칩니다.
  (되돌릴 수 없으니 미리 「데이터 백업」을 권장합니다.)
  · 특정 **주제만** 연습시키려면 「퀴즈 챌린지」에서 주제를 골라 출제하세요.
- **✎ 커스텀 퀴즈** — 선생님이 만든 문제를 파일이나 클립보드로 가져와,
  퀴즈 챌린지의 "커스텀 · 선생님 문제" 주제로 출제할 수 있습니다.
  문제 템플릿(JSON)도 이 화면에서 받을 수 있습니다.
  · **쉬운 문제 만들기** — `tools/editor.html` 파일을 두 번 눌러 열면(인터넷 불필요),
    빈칸을 채우는 방식으로 문제를 만들고 정답·해설·글자 수까지 확인한 뒤
    JSON으로 저장/복사할 수 있어요. 그 파일을 위 "가져오기"로 넣으면 됩니다.
- **🏆 수료증** — 학생별 학습 결과를 담은 인증서 화면. 캡처하거나 인쇄해
  과제·포트폴리오로 활용하세요.
- **퀴즈 챌린지** — 연속 정답, 오답 뒤 생각 질문, 결과 화면의 다음 학습 추천을
  활용해 수업 도입·정리 복습으로 짧게 운영할 수 있습니다.
- **⇄ 데이터 백업·복원** — 모든 슬롯·기록·설정을 한 파일로 저장하고 다른
  기기에서 복원합니다. 기기 교체나 학기말 보관에 쓰세요.

- **난이도**: 저학년 / 기본 / 고학년 — 문항 난이도와 안내 밀도가 함께 조정됩니다.
- **자막 속도**: 느림 / 보통 / 빠름
- **큰 글씨**, **색약 모드**, **읽어주기(TTS)**, **화면 효과 줄이기**(광과민성 배려)

## 5. 수업에 쓰는 흐름(예시)

1. 도입 — 오늘 다룰 주제를 한 문장으로 소개(예: "AI도 틀릴 수 있을까?").
2. 플레이 — 해당 스테이지를 15~20분 진행. 정답만 고르는 것보다
   **윤리 축 선택의 이유**와 위험한 선택의 결과를 말하게 합니다.
3. 멈춤 질문 — 스테이지 퍼즐 완료 뒤 나오는 **배움 조각**을 학생이 자기 말로
   바꿔 설명하게 합니다. "이 선택이 왜 안전했을까?", "다음에는 무엇을 확인할까?"
   처럼 행동 이유를 말하게 하면 단순 정답 암기를 줄일 수 있습니다.
4. 정리 — 대시보드에서 반 정답률과 윤리 축 히트맵을 보고, 수호자 일지의
   윤리 조각·윤리 이해도, 텍스트 리포트의 스테이지별 **배움 조각**과 마지막 결말 회고를 확인한 뒤
   약점 주제를 함께 토론합니다.
5. 확장 — `docs/차시별-활동지.md`의 토론·활동지를 활용합니다.

### 스테이지 1 관찰 포인트

경계마을 일대에서는 안내자와 표지 단서를 확인한 뒤 **데이터 발자국 숲**의 사진·집
위치·좋아하는 색 단서를 `공유 가능 / 동의 필요 / 공유 금지`로 나눕니다.
학생이 "내 정보인지", "친구의 동의가 필요한지", "공개하면 위험한지"를 말로
설명하게 하면 개인정보 보호 개념을 게임 행동과 바로 연결할 수 있습니다.

### 스테이지 2 관찰 포인트

햇살초원에서는 바람 언덕과 안개 습지의 안내자 단서를 확인한 뒤, 추천 카드를
`같은 의견 / 다른 의견 / 근거 확인`으로 나누어야 신호 탑터가 열립니다.
학생이 같은 의견만 반복해 고르면 미로가 입구로 되돌아가므로, "왜 반대 의견도
읽어야 하는지", "추천보다 근거가 필요한 순간은 언제인지"를 짧게 말하게 해 보세요.

### 스테이지 3 관찰 포인트

재깍사막에서는 나이·지역·기기 접근·언어 증거를 대표성 있게 모으는 장치를 풀어야 심판의
신전이 열립니다. 학생에게 "누가 데이터에서 빠졌는지", "이 결과가 특정 환경의
학생에게 불리하지 않은지"를 말하게 하면 알고리즘 편향과 공정성 개념을
저울 표시와 함께 확인할 수 있습니다.

### 스테이지 4 관찰 포인트

정지된 설원에서는 가짜 방송 현상을 관찰한 뒤 출처·날짜·원본·교차 확인 단서를
모아야 그림자성으로 이동합니다. 학생이 "그럴듯해 보이는 영상도 왜 바로
믿으면 안 되는지", "어떤 확인을 먼저 해야 하는지"를 말하게 하면 딥페이크와
합성 미디어 검증 습관을 확인할 수 있습니다.

### 스테이지 5 관찰 포인트

그림자성에서는 최종 종합 퍼즐 **책임의 코어**가 개인정보·추천 관점·공정성·검증·
인간 책임을 한 번씩 다시 묻습니다. 학생이 마지막 선택을 할 때 "AI가 골라도
왜 사람이 확인해야 하는지"를 말하게 하면 단순 정답보다 책임 있는 사용 태도를
평가할 수 있습니다.

전체 문항은 `docs/주제별-문제-목록.md`에서 한눈에 볼 수 있습니다(자동 생성).

## 6. 자주 묻는 질문

- **검은 화면만 나와요** — 처음 로딩에 실패한 경우 "다시 시도" 버튼이 보입니다.
  인터넷 연결을 확인하고 새로 고침하세요.
- **기록이 사라졌어요** — 같은 브라우저·기기에서만 저장됩니다(시크릿 모드 주의).
  미리 "데이터 백업"으로 파일을 받아 두면 안전합니다.
- **문제를 우리 반에 맞게 바꾸고 싶어요** — "커스텀 퀴즈"로 직접 추가하거나,
  개발자가 `src/data.js`의 `QUIZZES`를 수정한 뒤 `node tools/quizlist.js`로
  목록 문서를 다시 만들 수 있습니다.
